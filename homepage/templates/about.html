Sistema de Control de Inventario


| ¿Que es? |


Este es un software de gestión de inventario simple creado para los requisitos de una empresa. El software es capaz de realizar un seguimiento de las existencias entrantes y salientes con una factura generada dinámicamente. A cada proveedor se le proporciona un perfil donde puede ver el historial de sus transacciones con la empresa.


| Caracteristicas |


  • Gráfico interactivo para mostrar el stock disponible actual
  • Se mantienen los detalles del stock, como el stock disponible actual
  • Las transacciones entrantes se realizan a través de compras.
  • Las transacciones salientes se realizan a través de ventas.
  • Cada transacción realizada a través del software se registra
  • La factura se procesa dinámicamente y se puede imprimir
  • Se mantienen perfiles separados para los proveedores.
